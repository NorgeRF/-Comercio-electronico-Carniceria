# Directorio de Imágenes

Este directorio debe contener las siguientes imágenes:

## Imágenes requeridas:

1. **hero-bg.jpg** - Imagen de fondo para el hero section (recomendado: 1920x1080)
2. **products-bg.jpg** - Imagen de fondo para la página de productos
3. **orders-bg.jpg** - Imagen de fondo para la página de pedidos
4. **contact-bg.jpg** - Imagen de fondo para la página de contacto

## Imágenes de productos:

Crea subdirectorios para cada categoría:
- vacuno/
- cerdo/
- pollo/
- cordero/
- elaborados/
- mariscos/
- quesos/

## Nombres recomendados:
- vacuno-lomo.jpg
- cerdo-chuletas.jpg
- pollo-entero.jpg
- cordero-costillas.jpg
- elaborados-hamburguesas.jpg
- mariscos-salmon.jpg
- quesos-cabra.jpg

## Especificaciones:
- Formato: JPG o WebP
- Tamaño: Mínimo 800x600px
- Peso: Máximo 200KB por imagen
- Calidad: Buena compresión sin pérdida visible

## Si no tienes imágenes reales:

Puedes usar placeholders de:
- https://unsplash.com/ (gratis, alta calidad)
- https://placeholder.com/ (placeholders simples)
- https://loremflickr.com/ (imágenes temáticas)
